# projeto_crud
Um projeto que eu criei para treinar minhas habilidades de  logica de programação e testar também a criação de crud's do zero utilizando somente python para treinar a minha manipulação de dicionarios
Nele contem as funções basicas de um C.R.U.D:
C - Criei uma função para criar um usuário e guardar o nome dele em um diciodicionário, junto ao nome de usuário coloquei para o valor dele ser um id aleátorio de 0 a 10000 sem opção de repetição.
R - Para ler as informações cadastradas no dicionário criei uma função que consegue captar um usuário pelo seu nome e exibe seu nome e id no terminal.
U - Para atualizar as informações, apenas de nome por enquanto, criei duas funções que operam de maneira igual mas com entradas diferentes para localizar o usuário( uma entra com nome do usuário e a outra entra com o id).
D - Para deletar as informações do usuário contida no dicionário fiz igual a como eu fiz no update, criei duas funções iguais mas com meios de captação diferente.
